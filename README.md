\# AmbedkarGPT - AI Intern Assignment



A RAG (Retrieval Augmented Generation) Q\&A system based on Dr. B.R. Ambedkar's speeches.



\## Features

\- Question-Answering based on Ambedkar's speeches

\- Local LLM using Ollama with Mistral 7B

\- Simple text-based retrieval (embeddings fallback due to network restrictions)

\- Command-line interface



\## Setup Instructions



\### 1. Prerequisites

\- Python 3.8+

\- Ollama installed



\### 2. Install Ollama

Download and install Ollama from: https://ollama.ai/

Then pull the Mistral model:

```bash

ollama pull mistral
